# Python 3
## Virtual Environment
### Create new virtual Environment 
    
    python -m venv .\<your folder>

## Activate virtual Environment

    > .\Scripts\activate.bat
    $ .\Scripts\activate

## Leave vitual Environment 

    > deactivate
    $ deactivate

## Install packages (while virtual environment is active)

    > pip install <your package or file with dependencies>
    $ pip install <your package or file with dependencies>

## Drop a file with your dependencies (while virtual environment is active)

    > pip freeze > <your filename>
